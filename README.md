## 🚀 Using the tool features

The tool interface lets you control common design workflows. 

### Automate layer organization
Select your layers in the timeline and use the Organize button. The script sorts layers by type, names them based on your settings, and applies color labels automatically. This creates a clean project structure.

### Pipeline building
Link various project assets into a single logic flow. Use the Build Pipe tool to connect pre-compositions. This maintains consistency across different project files in your workflow.

### Batch rendering
Avoid manual export steps. Use the Batch Render button to add multiple compositions to the Render Queue. The tool applies your pre-set output module settings to every item, ensuring every render matches your quality requirements.

## ❓ Troubleshooting common issues

If the script panel does not appear in the Window menu, verify that you placed the file in the correct folder. Windows often hides the Program Files directory; ensure you have permission to modify files in that folder.

If the script displays an error when you click a button, check that all layers in your composition are unlocked. The script cannot move or rename locked layers. Unlock all layers before you run the command.

For issues with render settings, confirm that Adobe After Effects recognizes your export plugins. If the render fails, check your disk space and ensure your output destination folder is reachable by the software.
