## What it does

ae-motion-compiler is a motion graphics automation framework designed for Adobe After Effects CC 2024 and newer. It is built to reduce repetitive animation work by generating procedural keyframes, assembling expressions, and handling layer operations through script-based workflows.

The toolkit is intended for animators and motion designers who need reusable animation patterns, template-oriented sequences, and batch-style actions from one place. Its main focus is structured automation for keyframe generation, expression handling, and render-related tasks inside After Effects.

## Script Features

- Procedural keyframe generation for repeatable motion setups
- Expression-based animation control for dynamic timeline behavior
- Layer management tools for handling larger compositions
- Render automation support for batch-oriented output workflows
- Keyframe generator utilities for faster setup
- Expression builder tools for constructing scripted motion logic
- Batch automation features for repeated production tasks
- Template-based animation workflows for reusable project structures
